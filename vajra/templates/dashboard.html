How Vajra Detects Supply Chain Attacks

1

Download Both Sources

Fetch the .tar.gz from PyPI and the tagged source from GitHub for the same version.

2

Compare SHA-256 Hashes

Every file is hashed. Mismatches, extra files, or missing tags are flagged instantly.

3

AI Triage (Optional)

Flagged diffs are analyzed by Claude to classify threats: malicious, suspicious, or benign.
